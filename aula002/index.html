Estrutura do HTML e Meta ViewPort

Sobre o ViewPort:

Introduzido pela Apple, a tag Meta ViewPort serve para renderizar o conteúdo do site para dispositivos móveis. Essa tag ajuda na responsividade do site.

Os navegadores para celular utilizam uma "janela virtual", chamada viewport. Essa janela virtual que renderiza a página geralmente é maior do que a tela do dispositivo. Então, a resolução dessa janela, muitas vezes, é mais larga do que a largura do dispositivo.

Para resolver esse "problema" a Apple introduziu essa meta tag viewport, atribuindo o tamanho do viewport no atributo content, que no caso é o "device-width", ou seja, a largura do próprio dispositivo. Ou seja, desta forma, a janela virtual irá se encaixar na tela do dispositivo móvel.
