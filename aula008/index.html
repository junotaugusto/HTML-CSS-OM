As divs abaixo encontram-se umas dentro das outras, com tamanhos diferentes.

Como não foi especificado o posicionamento das caixas, elas ficaram uma em cima da outra, saindo da div principal. Pelo fato de se tratarem de elementos em bloco, elas quebram a linha. Uma forma de "resolver" isso seria inserir o display float left, como foi feito e se encontra a imagem agora.

Este é um método mais antigo. Atualmente temos o flex ou ridge que veremos mais à frente no curso. O princípio é entender que um site é estruturado com caixas dentro de caixas dentro de caixas...
