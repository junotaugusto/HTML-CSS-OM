Style e atributos ID e Class

Observações sobre IDs e Classes

Em HTML existem tags que possuem atributos que tem algum valor, ou atributos que não possuem valor nenhum. Nesses casos, o navegador insere um valor booleano a eles.

Os atributos mais "famosos" são o ID e CLASS.

O ID serve como um identificador da tag e eles possuem um valor, sempre entre aspas.

O identificador é único na página. Não podemos ter outra tag html com o mesmo identificador.

Ja as classes podem ter mais de uma.

Basicamente esplicou como selecionar determinadas tags do html usando a tag <style> dentro do cabeçalho.

Ele aplica as especificidades aos seletores que estiverem por último.
