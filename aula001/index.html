Estrutura básica do HTML

Algumas observações iniciais:

Tags possuem atributos inline.

O navegador tenta corrigir ao máximo qualquer tag ou algo "errado" que o desenvolvedor do site faz.

HTML não é case sensitive, ou seja, não importa se estiver com letra maiúscula ou minúscula.

Todas as outras tags são filhas da tag <html>.

A Tag HEAD:

A tag <head> é o cabeçalho do site e não aparece no documento. Ela trata de metadados do html, basicamente especificando o que a página html possui.

A tag de título da página (<title>) se encontra dentro do cabeçalho.

Outra tag importante é a <meta> que possui o atributo charset e que determina qual a codificação de caracteres que o usuário estará utilizando.
