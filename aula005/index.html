Tags Importantes e Semântica em Textos

Tags inline

Algumas tags, geralmente, contém outras tags. Isso se afirma quando a tag é um bloco. Ao inspecionar um documento, verifique na aba computed qual o display da tag. A característica de um bloco é que normalmente ele pega a largura inteira de um elemento e quebra uma linha.

Geralmente, dentro de um parágrafo, não podemos inserir nenhuma tag com display block. Apenas aceitando tags inline, como é o caso de <span>, <strong>, lembrando que a tag <strong> é semântica, ou seja, ela possui um significado, diferentemente de <b>, que apenas deixa o texto em negrito.

Com imagens, mesma coisa. Podemos inclusive colocar uma imagem dentro de um link, ou seja, a tag <a> aceita a tag <img> dentro dela conforme vemos na imagem abaixo.

No caso, o atributo alt refere-se ao texto alternativo da imagem, caso a imagem quebre no meio do caminho. Já o atributo title dá uma descrição para a imagem. Ao passar o mouse por cima, o usuário poderá ler a mensagem atribuída à imagem.

Imagens são elementos inline, o que quer dizer que caberia dentro de um parágrafo, por exemplo. Para deixá-las no centro da tela e quebrando a linha, por exemplo, é preciso transformá-las em bloco. Por isso, no CSS, adicione ao seletor o display: block e a margin: auto. Como os elementos em bloco ocupam toda a largura do html, ele irá centralizar a imagem, diferentemente dos elementos inline que ocupam apenas o próprio espaço.

Mesma coisa com a tag <a>.

Tags em bloco

As tags blockquote, code, hr entre outras são em bloco.

As tags SPAN e DIV são containeres genéricos.

O elemento HTML <span> é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ), ou para compartilhar valores de atributos como lang. Ele deve ser usado somente quando nenhum outro elemento semântico for apropriado. <span> é muito parecido com o elemento <div> , entretando <div> é um elemento de nível de bloco enquanto <span> é um elemento em linha.
